Hello Stealth...

Stealth is a Ruby framework for creating conversational (voice & chat) bots.

GitHub

Stealth was designed to bring all the good things about web development to the world of building bots. It sports an MVC architecture and it's convention of configuration philosophy is inspired by Ruby on Rails.

We can't wait to see the bots you make with Stealth.

Overview

Getting started with Stealth is easy. Our brief overview will run you through the basics.

Docs

Ready to build a bot of your own? Our documentation will guide you through the process.

Integrations

Stealth is extensible. All service and messaging integrations are split out into separate Ruby Gems.
